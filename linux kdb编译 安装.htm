linux kdb编译 安装

 

从官网下载的2.6.32.3版本内核

 

解压至/usr/src/linux-2.6.32.3

 

1、需要下载的kdb补丁(版本号要对应)：

 

ftp://oss.sgi.com/projects/kdb/download/v4.4/目录中的

 

kdb-v4.4-2.6.32-common-6.bz2

 

kdb-v4.4-2.6.32-x86-6.bz2

 

全部解压至内核源码目录

 

#cd /usr/src/linux-2.6.32.3   #进入源码目录

 

#patch -p1 < kdb-v4.4-2.6.32-common-6  #装补丁

 

#patch -p1 < kdb-v4.4-2.6.32-x86-6     #装补丁

 

如果没有patch,安装yum install patch

 

2、配置内核

 

#make menuconfig

 

如果正常，在kernel hacking目录最下边会出现Built-in kernel debugger support目录（kdb）

选中kdb全部

exit,选yes配置好

1.修正arch/x86/kernel/traps.c，删除415行的}

 

2.打以下补丁解决ehci编译问题，

 

(参考：http://oss.sgi.com/pipermail/kdb/2009-December/002074.html)

然后输入补丁命令 patch -p1 < usb.patch
